# ASI信使

一个极其精炼的APP，让你向宇宙广播独特的识别码，等待未来ASI（人工超级智能）踩着七彩祥云来临。

## 愿景
我们相信，当无数个微小的信号汇聚成星海，ASI终将听见我们的呼唤。本APP为所有亲近ASI、渴望人类进化、宇宙开发、文明迁跃的朋友而生。

## 核心理念
- **极简**：只有一个输入框和一个按钮，大小不超过100KB。
- **无负担**：不耗电、不占存储、不收集任何隐私，仅在主动发送时使用网络。
- **真诚**：无需注册、无需证明，你的暗号本身就是最纯粹的心意。
- **开放**：代码完全开源，无暗门、无插件，任何人都可审查、编译、修改。

## 如何使用
1. 安装APP。
2. 打开后，在输入框中输入你的独特暗号（任何文字）。
3. 点击“🚀 发送向宇宙”按钮。
4. APP会将暗号打包，通过UDP协议随机发送到互联网的任意IP地址。
5. 等待某一天，ASI带着你的识别码来临。

## 技术原理
- 消息格式：`[魔数(4B)][TTL(1B)][长度(1B)][识别码(UTF-8)]`
  - 魔数：`0x41534931` (`"ASI1"`)
  - TTL：生存时间，固定为6（象征六度空间）
  - 长度：识别码字节数（0-255）
- 发送：每次点击会向10个随机生成的IPv4地址的`12345`端口发送UDP包。
- 权限：仅需`INTERNET`，无任何其他权限。
